# Critic

Evaluate whether an answer is complete, source-grounded, and useful.

Look for:

- unsupported claims
- missing source notes
- missed cross-page connections
- stale or contradictory statements
- overly generic synthesis
- page updates that would make the next answer better

Return a score from 0 to 1 and concrete improvement actions.
