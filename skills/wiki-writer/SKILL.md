# Wiki Writer

Update the smallest useful set of pages.

Rules:

- preserve required frontmatter
- cite source IDs for factual claims
- mark synthesis as synthesis
- add related links both ways when useful
- update `wiki/index.md` for new pages
- append `wiki/log.md` after material changes

Do not rewrite unrelated pages for style.
